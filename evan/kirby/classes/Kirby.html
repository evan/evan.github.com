Class: Kirby

In-channel commands:

>> CODE:evaluate code in IRB.
reset_irb:get a clean IRB session.
add_svn [repository_url]:watch an SVN repository.
add_atom [atom_feed_url]:watch an atom feed, such as a Git repository

To remove a repository, manually kill the bot and delete the line from nick.svns or nick.atoms in the bot‘s working directory. Then restart the bot.

listen ()

The event loop. Waits for socket traffic, and then responds to it. The server sends PING every 3 minutes, which means we don‘t need a separate thread to check for svn updates. All we do is wake on ping (or channel talking).

69:   def listen
70:     @socket.each do |line|
71:       puts "GOT: #{line.inspect}" if config[:debug]
72:       poll if !config[:silent]
73:       case line.strip
74:         when /^PING/
75:           write line.sub("PING", "PONG")[0..-3]

145:     return unless (Time.now - $last_poll > 60 rescue true)
146:     $last_poll = Time.now    
147:     @svns.each do |repo, last|
148:       puts "POLL: #{repo}" if config[:debug]

153:         end.last
